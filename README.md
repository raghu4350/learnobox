# NN Project

## Project Description
The NN Project is a full-stack application designed to demonstrate the capabilities of neural networks in solving complex problems. The project consists of both backend and frontend components, allowing users to interact with the neural network models seamlessly.

## Installation Instructions
To set up the project locally, follow these instructions:

### Backend Installation
1. Navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

### Frontend Installation
1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

## Usage Guide

### Backend
To run the backend server:
1. Ensure you are in the backend directory.
2. Start the server:
   ```bash
   npm start
   ```
3. Access the API at `http://localhost:5000`.

### Frontend
To run the frontend application:
1. Ensure you are in the frontend directory.
2. Start the application:
   ```bash
   npm start
   ```
3. Access the application at `http://localhost:3000`.
